<b>Installionshinweise</b>
<br/>
Bitte folgende Packages im Projekt installieren:<br/>
- meteor add coffeescript
- meteor add fourseven:scss
- meteor add kadira:flow-router
- meteor add kadira:blaze-layout
- meteor add vue:vue

mit folgendem Befehl kann man alle auf einmal installieren:<br />
<pre> meteor add coffeescript fourseven:scss kadira:flow-router kadira:blaze-layout vue:vue</pre>
